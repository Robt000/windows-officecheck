# 企业信息安全审计系统

## 项目简介

企业信息安全审计系统是一款专为企业内部信息安全管理设计的工具，主要用于扫描、检测和管理企业内部的Office文档，识别可能存在的敏感信息，帮助企业防范信息泄露风险。

## 系统架构

系统由客户端和服务器端两部分组成：

- **客户端**：运行在员工电脑上的扫描工具，负责扫描本地文档并将结果上传至服务器
- **服务器端**：接收、存储和分析客户端上传的数据，提供Web界面进行数据查看和管理

## 主要功能

### 客户端功能

- 全盘扫描：自动扫描计算机所有盘符中的Office文档
- 文档分析：提取文档元数据和内容，识别敏感信息
- 数据加密：使用加密算法保护扫描结果
- 离线导出：支持将扫描结果导出为加密文件，便于后续导入服务器

### 服务器端功能

- 数据接收：接收客户端上传的扫描结果
- 数据分析：对文档信息进行分类和统计
- 风险评估：根据敏感信息识别结果评估风险等级
- 数据可视化：通过图表直观展示统计结果
- 用户管理：支持多用户访问和权限控制

## 技术栈

- **客户端**：Python, Tkinter, Cryptography
- **服务器端**：Flask, SQLAlchemy, SQLite, Bootstrap

## 使用方法
### 客户端使用
1. 启动客户端程序
2. 输入员工姓名和部门信息
3. 点击"开始扫描"按钮开始扫描
4. 扫描完成后，数据会自动上传至服务器
5. 如需离线使用，可点击"导出数据"按钮将结果保存为加密文件
### 服务器端使用
1. 访问服务器Web界面 (默认地址: http://localhost:5000 )
2. 使用管理员账号登录系统
3. 在仪表盘查看统计数据和风险评估结果
4. 通过"会话管理"查看各次扫描会话详情
5. 通过"文档管理"查看所有扫描到的文档信息
## 安全说明
- 所有客户端与服务器之间的通信均经过加密
- 导出的数据文件使用AES-256算法加密
- 服务器存储的敏感信息使用加密存储
